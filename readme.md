## 基于electron的快捷桌面端软件生成demo

此案例以[H5-Dooring](http://h5.dooring.cn)零代码平台为例, 通过项目的安装和打包, 可以快速生成一个桌面端软件.

## 本地使用

1. 安装依赖
```
yarn
```

2. 本地启动

```
yarn start
```

3. 打包

```bash
# 打包window版本
build:win

# 打包mac版本
build:mac
```

同时我的朋友开发了一款线上生成桌面端软件的工具 `webdesk`, 大家也可以参考学习一下, 

在线地址: https://webdesk.pigjs.com/builder
